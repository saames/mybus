**1. Crie o ambiente virtual**

`python3 -m venv venv`

**2. Acesse o ambiente virtual**

- Windows: `.venv\Scripts\activate`
- Linux: `source .venv/bin/activate`

**3. Instale as bibliotecas necessárias:**

`pip install -r requirements.txt`
